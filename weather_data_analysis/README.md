# Weather Data Analysis

This project analyzes a weather dataset to understand patterns in temperature and humidity.

## Project Objectives
- Load weather dataset using Python
- Analyze temperature and humidity trends
- Create visualizations
- Identify patterns in weather data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data overview and statistical summary
- Temperature trend analysis
- Humidity trend analysis
- Data visualization
- Pattern identification

## Insights
- Temperature shows seasonal trends over time.
- Humidity fluctuates based on weather conditions.
- Some features like "Loud Cover" contained constant values and were removed.
